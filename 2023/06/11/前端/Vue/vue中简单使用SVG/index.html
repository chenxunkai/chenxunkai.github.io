认识svg

SVG 全称是可缩放矢量图，体积较小，且不管放大多少倍都不会失真

svg简单使用

  • 前往阿里图标库 https://www.iconfont.cn/ ，随便找个简单点的图标点击 下载-> 复制SVG代码,到vue中使用，运行后会出现打勾的图标

以上代码中,

  • width 和 height 指定元素的宽度和高度，如果不指定默认大小是 300 像素（宽） x 150 像素（高）
  • viewBox 属性有4个值，分别是左上角的横坐标和纵坐标、视口的宽度和高度
  • path标签，是用来定义形状的通用元素，所有的基本形状都可以用 path 来创建。fill属性，可设置svg的填充色，若要设置svg图标颜色需用此属性，此外还有stroke属性：描边色，stroke-width属性：边框宽度

svg其他标签

  • path 标签
    用来定义形状的通用元素，所有的基本形状都可以用 path 来创建,此外还有绘制圆形的circle标签

  • use 标签
    复制使用绘制好的svg图标,使用前需在要复制的svg图标设置id，使用时在href属性指定所要复制的节点,以下代码中，x 属性和 y 属性是 use标签 左上角的坐标

  • g 标签
    用于将多个形状组成一个组（group）

  • defs 标签
    使用该标签后内部的代码不会显示，同g标签一起使用，可在svg内部当成模板使用

  • symbol 标签
    用来对元素进行分组，它不会被直接显示，可当成模板使用,可在其他地方使用
