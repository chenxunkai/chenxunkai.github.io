vue项目将SVG封装成svgIcon组件

发表于2023-06-11|更新于2023-06-11|svg
|字数总计:816|阅读时长:4分钟|阅读量:

前置知识： vue中简单使用SVG

创建vue项目

  • 使用vue create xxx创建vue项目，创建步骤省略

安装svg-sprite-loader, svgo@1.2.0

创建下载svg

  • src -> icon文件夹 -> svg文件夹(放svg图标)
  • src -> icon文件夹 -> index.js
  • src -> icon文件夹 -> svgo.yml(后面用于配置svgo)
  • 前往阿里图标库 https://www.iconfont.cn/ ，随便找个简单点的图标点击 下载-> SVG下载,将svg放到svg文件夹,重命名成英文

vue.config.js配置svg

创建svgIcon组件并导入

  • src -> components文件夹 -> svgIcon文件夹 -> index.vue

  • 全局注册组件，src -> icon文件夹 -> index.js

  • 导入组件，src -> main.js

  • 使用svgIcon组件

svgo工具

使用svgo对 src/icon/svg 目录下的 SVG 文件进行优化处理

  • src -> icon文件夹 -> svgo.yml
  • package.json增加svgo脚本
  • 使用svgo工具

运行命令后会自动优化svg
